# comment-no-empty

Disallow empty comments.

```css
    /* */
/** â†‘
 * Comments like this */
```

**Caveat:** Comments within *selector and value lists* are currently ignored.

## Options

### `true`

The following patterns are considered violations:

```css
/**/
```

```css
/* */
```

```css
/*

 */
```

The following patterns are *not* considered violations:

```css
/* comment */
```

```css
/*
 * Multi-line Comment
**/
```
