# selector-no-universal

Disallow the universal selector.

```css
    * {}
/** â†‘
 * This selector */
```

## Options

### `true`

The following patterns are considered warnings:

```css
* {}
```

```css
*, .foo {}
```

```css
* > [foo] {}
```

The following patterns are *not* considered warnings:

```css
a {}
```

```css
.foo {}
```

```css
[foo] {}
```

```css
#foo {}
```

```css
.bar > #foo {}
```
